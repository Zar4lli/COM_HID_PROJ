# COM_HID_PROJ

Проект для автоматизации действий на ПК с использованием YOLO и HID-устройства.

## Состав проекта

### COM_HID_CLIENT_PYTHON

Python-клиент с графическим интерфейсом.

Функции:

* подключение к HID-устройству через COM-порт;
* отправка команд управления мышью;
* поиск объектов на экране через YOLO;
* автоматическое наведение курсора;
* выполнение кликов по найденным объектам.

### COM_HID_DEVICE

Прошивка HID-устройства.

Устройство определяется компьютером как обычная USB-мышь и выполняет команды, полученные от Python-клиента.

### YOLO_DETECT

Скрипты для детекции объектов на экране.

Используется:

* Ultralytics YOLO;
* MSS для захвата экрана;
* OpenCV для обработки изображений.

Поддерживается:

* поиск объектов по классу;
* выбор объекта с максимальной уверенностью;
* поиск ближайшего объекта к курсору;
* поиск ближайшего объекта к центру экрана.

### YOLO_TRAIN

Скрипты для обучения и подготовки моделей.

## Установка

Создать виртуальное окружение:

```bash
python -m venv .venv
```

Активировать:

```bash
.venv\Scripts\activate
```

Установить зависимости:

```bash
pip install ultralytics opencv-python mss pyserial numpy torch
```

## Запуск

```bash
python COM_HID_CLIENT_PYTHON/main.py
```
